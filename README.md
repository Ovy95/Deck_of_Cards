# Deck

Covering some of the basics in Golang, this project is a sample deck of cards. Created by following a Udemy course to learn from.<br>

Features
- Dealing out x amount of cards using Int value
- Shuffle the deck 
- save the deck as a file
- load the deck file

Topics covered 
- Imports
- Custom types 
- Functions
- Receiver functions
- Saving files 
- Loading saved files
- Testing in Go
